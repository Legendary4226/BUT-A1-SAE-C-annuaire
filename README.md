Vous avez plusieurs commandes à disposition, les voici :

- add : Permet d'ajouter un client.
- alter : Permet de modifier un client existant.
- delete : Permet de supprimer un client existant.
- missingInfo : Permet d'afficher les clients dont des informations sont manquantes.
- loadFile : Permet de charger le fichier que vous souhaitez éditer.
- filter : Permet de rechercher/filtrer un champs en fonction d'une valeur dans le fichier.
- display : Permet d'afficher la totalité du contenu du fichier.
- save : Permet d'enregistrer les modifications apportées au fichier.
- tri : Tri la liste selon un champ.\n\n"

Pour connaitre les détails des commandes tapez la commande dans le format suivant :
help -cmdName (nom de la commande)


/!\ Pour pouvoir exécuter ces commandes, il est impératif de charger le fichier que vous allez manipuler, pour cela faites la commande suivante :
loadFile -path (chemin du fichier)
